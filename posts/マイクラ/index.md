# 🎮 ブロッククラフト - マインクラフト風ゲーム

ブラウザで動作する3Dボクセルゲームです。Three.jsを使用して、マインクラフト風の世界を実装しました。

## 🎯 機能

### ✅ 実装済み機能

- **3D地形生成**: Perlin Noise風のアルゴリズムで自然な地形を自動生成
- **6種類のブロック**: 草、土、石、木材、葉、砂
- **ブロックの配置と破壊**: 右クリックで配置、左クリックで破壊
- **FPSスタイルの移動**: WASDキーでの移動とマウスでの視点変更
- **インベントリシステム**: ホットバーで素早くブロックを切り替え
- **木の自動生成**: ランダムに木が生成される
- **リアルな照明**: 環境光と指向性ライト
- **物理演算**: 重力とジャンプ機能
- **霧エフェクト**: 遠方の描画距離制限

## 🎮 操作方法

### 基本操作
- **W / A / S / D**: 移動（前後左右）
- **スペース**: ジャンプ
- **マウス移動**: 視点変更（ポインターロック中）
- **左クリック**: ブロック破壊
- **右クリック**: ブロック配置

### ブロック選択
- **1〜6キー**: ブロックタイプを切り替え
- **インベントリクリック**: マウスでブロック選択

## 📦 ブロックの種類

1. **草ブロック** (Grass): 地表の基本ブロック
2. **土** (Dirt): 地表の下層ブロック
3. **石** (Stone): 地下の基本ブロック
4. **木材** (Wood): 木の幹として使用
5. **葉** (Leaves): 木の葉として使用
6. **砂** (Sand): 水辺に生成

## 🛠️ 技術詳細

### 使用技術
- **Three.js**: 3Dグラフィックスライブラリ
- **Vanilla JavaScript**: フレームワークなしで実装
- **HTML5/CSS3**: UIとレイアウト

### 主な実装
```javascript
// 地形生成アルゴリズム
- SimplexNoise クラスによるノイズ生成
- 2オクターブのノイズで自然な起伏を生成
- バイオームに応じたブロック配置

// ゲームループ
- requestAnimationFrame による滑らかな描画
- デルタタイムを使用した物理演算
- Raycaster によるブロックの選択と操作

// 最適化
- 視錐台カリング（Three.js自動）
- オブジェクトプーリング対応
- レンダリング距離制限
```

## 🚀 起動方法

1. `game.html` をブラウザで開く
2. 「ゲーム開始」ボタンをクリック
3. ポインターロックを許可
4. ゲームを楽しむ！

## 🎨 カスタマイズ

### ブロックの追加方法

`game.js` の `BLOCK_TYPES` オブジェクトに新しいブロックを追加：

```javascript
const BLOCK_TYPES = {
    // 既存のブロック...
    CUSTOM: { 
        id: 'custom', 
        name: 'カスタムブロック', 
        color: 0xFF00FF 
    }
};
```

### 地形生成のカスタマイズ

`generateWorld()` 関数内でノイズパラメータを調整：

```javascript
const height = Math.floor(
    noise.noise2D(worldX * 0.05, worldZ * 0.05) * 8 + // 大きな起伏
    noise.noise2D(worldX * 0.1, worldZ * 0.1) * 4 +   // 細かい起伏
    10 // 基準高度
);
```

## 📊 パフォーマンス

- **推奨環境**: モダンブラウザ（Chrome, Firefox, Edge）
- **描画範囲**: 64ブロック
- **チャンクサイズ**: 32x32
- **目標FPS**: 60fps

## 🐛 既知の問題

- ブロック数が多くなると処理が重くなる可能性があります
- モバイルデバイスでは操作が制限されます
- 一部のブラウザでポインターロックが正常に動作しない場合があります
