# 🎮 VoxelCraft - マインクラフト風ブロック建築ゲーム

## 📖 概要

VoxelCraftは、ブラウザで動作するマインクラフト風の3Dブロック建築ゲームです。Three.jsを使用して実装されており、パーリンノイズによる自然な地形生成、物理演算、複数のブロックタイプなど、中級レベルの機能を搭載しています。

## ✨ 主な機能

### 🌍 ワールド生成
- **パーリンノイズによる地形生成**: 自然で多様な地形（丘、平原、水域）
- **チャンクシステム**: 効率的な描画距離管理（4チャンク範囲）
- **複数バイオーム**: 草原、砂浜、水域などの地形
- **自然な木の生成**: ランダムに配置される木々

### 🎨 ブロックシステム
- **10種類のブロック**:
  - 🟢 草ブロック
  - 🟤 土
  - ⚪ 石
  - 🟫 木材
  - 🟩 葉
  - 🟡 砂
  - 💧 水（半透明）
  - 💎 ガラス（透明）
  - 🪵 板材
  
- **リアルなテクスチャ**: Canvas APIで生成されるプロシージャルテクスチャ
- **ブロックごとの特性**: 透明度、テクスチャバリエーション

### 🎮 プレイヤーシステム
- **滑らかな移動**: WASD キーによる8方向移動
- **物理演算**: 重力、衝突検出、ジャンプ
- **走行機能**: Shiftキーで高速移動
- **一人称視点**: マウスで360度自由に視点変更

### 🛠️ 建築システム
- **ブロック破壊**: 左クリックで素早く破壊
- **ブロック設置**: 右クリックで選択したブロックを配置
- **レイキャスト**: 最大10ブロック先まで正確に選択
- **ホットバー**: 1-9キーまたはクリックで素早くブロック切り替え

### 🖥️ ユーザーインターフェース
- **スタート画面**: ゲーム開始時の美しいグラデーション画面
- **クロスヘア**: 中央の照準マーカー
- **ホットバー**: 画面下部のブロック選択UI
- **デバッグモード**: F3キーでFPS、座標、チャンク数などを表示
- **レスポンシブデザイン**: モバイルにも対応

### 🌤️ 環境エフェクト
- **スカイボックス**: 明るい青空
- **霧効果**: 遠景の自然なフェード
- **影システム**: リアルタイムシャドウマッピング
- **ライティング**: 環境光＋太陽光

## 🎯 操作方法

### 基本操作
- **W / A / S / D**: 前後左右に移動
- **スペース**: ジャンプ
- **Shift**: 走る（移動速度1.5倍）
- **マウス移動**: 視点変更

### ブロック操作
- **左クリック**: ブロックを破壊
- **右クリック**: ブロックを設置
- **1〜9キー**: ホットバーのブロック選択
- **ホットバークリック**: マウスでブロック選択

### システム
- **F3**: デバッグ情報の表示/非表示
- **ESC**: ポインターロック解除（ポーズ）

## 🏗️ 技術仕様

### アーキテクチャ
```
VoxelCraft/
├── index.html    - メインHTMLファイル（UI、スタイル）
├── game.js       - ゲームエンジン（1000行以上）
└── index.md      - このREADME
```

### 主要技術
- **Three.js r160**: 3Dレンダリングエンジン
- **パーリンノイズ**: 自然な地形生成アルゴリズム
- **チャンクシステム**: メモリ効率的なワールド管理
- **レイキャスティング**: 正確なブロック選択
- **コリジョン検出**: AABB（軸平行境界ボックス）方式
- **Canvas API**: プロシージャルテクスチャ生成

### パフォーマンス最適化
- **チャンク遅延読み込み**: 必要なエリアのみ生成
- **メッシュ結合**: 1チャンク1メッシュで描画コール削減
- **自動アンロード**: 遠いチャンクの自動削除
- **LOD準備**: 将来的な詳細度管理の基盤

### ゲームパラメータ
```javascript
CHUNK_SIZE = 16          // チャンクの横幅（ブロック数）
CHUNK_HEIGHT = 128       // ワールドの高さ
RENDER_DISTANCE = 4      // 描画距離（チャンク数）
GRAVITY = -32            // 重力加速度
JUMP_FORCE = 12          // ジャンプ力
MOVE_SPEED = 8           // 通常移動速度
SPRINT_SPEED = 12        // 走行速度
PLAYER_HEIGHT = 1.8      // プレイヤー身長
```

## 🎨 実装の特徴

### 1. 高度な地形生成
- **オクターブノイズ**: 4オクターブで自然な起伏
- **バイオーム判定**: 高度に応じた地形タイプ
- **水域生成**: 標高40以下は自動的に水で満たされる
- **構造物生成**: 木が自然に配置される

### 2. リアルなテクスチャ
- **プロシージャル生成**: 実行時に動的生成
- **ノイズパターン**: 各ブロックに固有の模様
- **マルチフェイステクスチャ**: 草ブロックは上下側面で異なる
- **ピクセルアート風**: NearestFilterでクリスピーな見た目

### 3. 物理演算
- **3軸独立コリジョン**: X、Y、Z軸を個別に判定
- **地面接触判定**: ジャンプ可否の正確な検出
- **速度減衰**: 慣性のあるリアルな動き
- **AABB衝突**: 円柱近似によるプレイヤー判定

### 4. 効率的なメッシュ生成
- **カリング**: 隠れた面は生成しない
- **マテリアルグループ**: 効率的な描画バッチ
- **ダイナミックリビルド**: ブロック変更時のみ再生成
- **インデックスバッファ**: 頂点の再利用

## 🚀 使い方

### 開始方法
1. ゲームページを開く
2. 「ゲームを始める」ボタンをクリック
3. ポインターがロックされたら操作開始

### 建築のヒント
- **整地**: まず地形を平らにすると建築しやすい
- **材料選択**: ホットバーで色々なブロックを試す
- **高所作業**: ジャンプしながらブロックを足元に設置
- **水辺**: 水ブロックは半透明で美しい

### デバッグ
- **F3キー**: パフォーマンス情報を表示
  - FPS（フレームレート）
  - プレイヤー座標
  - ロード済みチャンク数
  - ワールド内のブロック総数

## 🐛 既知の問題

- チャンク境界でのブロック更新が遅延する場合がある
- 高速移動時に稀にブロックをすり抜けることがある
- モバイルでの操作は完全にはサポートされていない

## 💡 パフォーマンスTips

- **描画距離**: RENDER_DISTANCEを下げると軽くなる
- **チャンク高さ**: CHUNK_HEIGHTを下げるとメモリ節約
- **影**: 重い場合はrenderer.shadowMap.enabledをfalseに
- **霧**: 遠景の霧で描画負荷を視覚的に軽減
